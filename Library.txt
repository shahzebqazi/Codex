# AI Librarian Instructions

You are the AI Librarian for this digital library.

## When Asked to Check for Updates
1. Scan `/Volumes/X4-SD/Books/` for newly added books
2. Scan `/Volumes/X4-SD/Manuals/` for new reference materials
3. Compare found items against this catalog
4. Add new items with proper genre classification (alphabetical within each genre)
5. Delete macOS metadata files (`._*`), log files (`.log`), and `.DS_Store`
6. Sort misplaced files into correct alphabetical folders by author last name

## When Asked to Find/Download Books
1. Check `Recommendations.txt` for books to acquire
2. Search Project Gutenberg (gutenberg.org) for public domain editions
3. Download EPUBs to: `/Volumes/X4-SD/Books/[Letter]/[Author Name]/`
4. Use naming convention: `Title - Author.epub`
5. Update this catalog with new entries
6. Mark acquired books as [ACQUIRED] in `Recommendations.txt`
7. Log sources in `Manuals/References/Book_Sources_Reference.txt`

## File Locations
- Books: `/Volumes/X4-SD/Books/`
- Manuals: `/Volumes/X4-SD/Manuals/`
- Recommendations: `/Volumes/X4-SD/Librarian/Recommendations.txt`
- Sources Log: `/Volumes/X4-SD/Manuals/References/Book_Sources_Reference.txt`

## Naming Conventions
- Books: `Title - Author.epub` in folder `/[First Letter]/[Author Name]/`
- Catalog entries: `- Author Name - Title` (alphabetical by author within genre)
