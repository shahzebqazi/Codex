# Codex skills (Cursor plugin format)

These skills mirror [.ai/skills/](../.ai/skills/) in Cursor’s plugin skill format (YAML frontmatter + `SKILL.md`). They expose the same capabilities so Cursor and other plugins can discover and invoke them via the Skill tool.

- **Source of truth:** [.ai/skills/](../.ai/skills/) — edit there; this directory can be regenerated.
- **Structure:** One folder per skill (e.g. `agents/aider/`, `vcs/jj/`) with a single `SKILL.md` (name, description, body).
