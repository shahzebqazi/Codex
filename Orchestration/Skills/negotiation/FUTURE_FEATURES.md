# Negotiation skill — prompts for future features

Use these prompts when extending or designing additional features for the negotiation skill.

- **Multi-round protocols:** "Design a multi-round negotiation protocol where each agent can propose, accept, or counter-propose with a bounded number of rounds and a clear termination condition (agreement or timeout)."
- **Concession strategies:** "Define concession strategies (e.g. time-based, utility-based) that agents can use when scaling negotiation to many participants; include backoff and fairness constraints."
- **Facilitator API:** "Specify an API or interface for a facilitator agent: conflict detection, round management, proposal aggregation, and outcome emission."
- **Integration with other skills:** "Describe how negotiation can hand off to consensus (e.g. for ratification) or to priority (e.g. when negotiation deadlocks and an authority must decide)."
- **Negotiation under uncertainty:** "Extend the negotiation skill to handle incomplete information and probabilistic beliefs; define how agents share and update beliefs during negotiation."
