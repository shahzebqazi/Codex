# Skill: negotiation

**Keyword:** `negotiation` (and phrases like "resolve by negotiation", "negotiate conflict", "progressive negotiation")

**Effect (deterministic):**

1. Apply **negotiation-based conflict resolution** among distributed AI agents or components: detect goal or resource conflicts, then run a structured negotiation protocol (e.g. progressive negotiation, stepwise concessions, minimal backtracking).
2. Use or designate a **facilitator/coordinator** to orchestrate the protocol, surface proposals and counter-proposals, and converge on an agreed plan or resource allocation.
3. Emit the **negotiated outcome** (agreed actions, allocations, or shared plan) in the conversation or to the harness; do not write files unless the user or harness explicitly requests a persisted artifact.

**When to apply:** User or orchestrator requests conflict resolution via negotiation; goal conflicts or resource contention among agents; heterogeneous cooperating agents need a coordinated agreement without a single authority.

**Context (distributed AI):** Suited to goal conflicts and resource allocation where agents can make concessions and iteratively improve on a shared solution.
