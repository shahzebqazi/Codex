# Argumentation skill — prompts for future features

Use these prompts when extending or designing additional features for the argumentation skill.

- **Merge operators:** "Define and document belief-merge operators (e.g. majority merge, distance-based merge, integrity constraints). For each, specify input format (belief bases), output (merged base), and properties (e.g. consistency, fairness)."
- **Argumentation semantics:** "Specify supported semantics (e.g. grounded, preferred, stable) for abstract argumentation frameworks; include how to derive arguments and attacks from agent claims and how to output accepted conclusions."
- **Structured arguments:** "Support structured argumentation (premises, conclusion, type): define schema and how structure affects attack relation and semantics."
- **Uncertainty:** "Extend to probabilistic or weighted beliefs and arguments; define how weights propagate and how to report accepted conclusions with confidence or probability."
- **Integration with consensus:** "Describe when to use argumentation to prepare a candidate set of conclusions and then use consensus or voting to select among them in a hybrid pipeline."
