# Docs site content

This directory is the source for the GitHub Pages site. The published **home page** is a glossary only: no code snippets. Other pages (if added) may include code examples.

- **Home page:** `index.md` — Technical terms for AI users (definitions and links; no code).
- **Rebase and setup:** `REBASE_STRATEGY.md` (in-repo only; excluded from site), `PAGES_SETUP.md` (maintainer instructions; excluded from site).

To change the live site, edit the Markdown here and merge to `main`; the workflow will build and deploy.
